# ea-fc-fifa-team-generator

## Install Project

```sh
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## Virtual Environment.

### Activate (Mac)

```sh
source venv/bin/activate
```

### Activate (Mac)

```sh
venv\Scripts\activate
```

### Install Packages

```sh
pip [Command]
```

### Deactivate (Mac)

```sh
deactivate
```
